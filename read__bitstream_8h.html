Detailed Description

this library handles the reading from an MSB-first bitstream

This API consists of small unitary functions, which must be inlined for best performance. Since link-time-optimization is not available for all compilers, these functions are defined into a .h to be included.

Start by invoking bit_init_decoder(). A chunk of the bitstream is then stored into a local register. The local register size is 64 bits. You can then retrieve bit-fields stored into the local register. The local register is explicitly reloaded from the memory with the bit_refill() function. A reload guarantees a minimum of 57 bits in the local register if the returned status is BIT_UNFINISHED. Otherwise, it can be less than that, so proceed accordingly. Checking if bit_decoder has reached its end can be performed with bit_end_of_stream().
