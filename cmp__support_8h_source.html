242  * [bit 2] model_value_err; The model_value parameter is not set correctly
243  * [bit 3] cmp_par_err; The spill, golomb_par combination is not set correctly
244  * [bit 4] ap1_cmp_par_err; The ap1_spill, ap1_golomb_par combination is not set correctly (only HW compression)
245  * [bit 5] ap2_cmp_par_err; The ap2_spill, ap2_golomb_par combination is not set correctly (only HW compression)
246  * [bit 6] mb_err; Multi bit error detected by the memory controller (only HW compression)
247  * [bit 7] slave_busy_err; The bus master has received the "slave busy" status (only HW compression)
248  * [bit 8] slave_blocked_err; The bus master has received the slave blocked status (only HW compression)
249  * [bit 9] invalid address_err; The bus master has received the invalid address status (only HW compression)
