set up the configuration parameters for an auxiliary science data compression

Note
auxiliary compression data types are: DATA_TYPE_OFFSET, DATA_TYPE_BACKGROUND, DATA_TYPE_SMEARING, DATA_TYPE_F_CAM_OFFSET, DATA_TYPE_F_CAM_BACKGROUND
not all parameters are needed for the every auxiliary compression data type
Parameters
cfgpointer to a compression configuration (created with the cmp_cfg_icu_create() function)
cmp_par_meanmean compression parameter
spillover_meanmean spillover threshold parameter
cmp_par_variancevariance compression parameter
spillover_variancevariance spillover threshold parameter
cmp_par_pixels_erroroutlier pixels number compression parameter
spillover_pixels_erroroutlier pixels number spillover threshold parameter
Returns
0 if parameters are valid, non-zero if parameters are invalid
