To compress data, first create a compression configuration with the rdcu_cfg_create() function. Then set the different data buffers with the data to compressed, the model data and the RDCU SRAM addresses with the rdcu_cfg_buffers() function. Then set the imagette compression parameters with the rdcu_cfg_imagette() function. Finally, you can compress the data with the RDCU using the rdcu_compress_data() function.
