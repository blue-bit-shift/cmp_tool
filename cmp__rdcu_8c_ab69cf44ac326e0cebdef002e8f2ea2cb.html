const struct cmp_info *  last_info 
)

compressing data with the help of the RDCU hardware compressor; read data from the last compression before starting compression

Parameters
cfgconfiguration contains all parameters required for compression
last_infocompression information of last compression run
Note
when using the 1d-differencing mode or the raw mode (cmp_mode = 0,2,4), the model parameters (model_value, model_buf, rdcu_model_adr) are ignored
the overlapping of the different rdcu buffers is not checked
the validity of the cfg structure is checked before the compression is started
Returns
0 on success, error otherwise
