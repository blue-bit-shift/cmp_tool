subtract the model from the data, encode the result and put it into bitstream, for encoding outlier use the multi escape symbol mechanism

Parameters

modelmodel of the data (0 if not used)
stream_lenlength of the bitstream in bits
setuppointer to the encoder setup
Returns
the bit length of the bitstream with the added encoded value on success; negative on error, CMP_ERROR_SMALL_BUF if the bitstream buffer is too small to put the value in the bitstream
Note
no check if the data or model are in the allowed range
no check if the setup->spillover_par is in the allowed range
