create a complete package from header and payload data including CRC8

Note
this is a helper function to generate complete binary RMAP packet dumps
Parameters
blobthe blob buffer; if NULL, the function returns the needed size
[in]cmdan rmap command buffer
[in]cmd_sizethe size of the rmap command buffer
[in]non_crc_bytesleading bytes in the header not path of the CRC
[in]dataa data buffer (may be NULL)
[in]data_sizethe size of the data buffer (ignored if data is NULL)
Note
data_size must be a multiple of 4
this function will convert all data to big endian as 32 bit words
Returns
the size of the blob or 0 on error
